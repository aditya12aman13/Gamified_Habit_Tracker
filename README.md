# Gamified Habit Tracker

A Django web app that helps users build habits, quit bad habits, track progress and earn rewards.

## Features
- Create habits
- Qit unwanted habits
- Track daily/weekly progress
- Earn points and unlock badges
- Dashboard with progress bars

## Tech Stack
- Django 5
- HTML + CSS (Bootstrap/Tailwind)
- SQLite

# Installation
1. Clone the repo
2. Create virtual environment: `python -m venv venv`
3. Activate venv: `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Run migrations: `python manage.py migrate`
6. Start server: `python manage.py runserver`
